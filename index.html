Projects


Software Package for Engineering Drawing
Jan 2018-March 2018

  • The software package is developed in C++ and caters to the basic functionalities, required for Engineering Drawing. The project was developed as a part of the course Design Practices (COP290), under Prof. Subhashis Banerjee.
  • The implemented functionalities were projection of a 3D Object (without curvilinear edges) on any plane and 3D object generation from the Orthographic Views.
  • GUI with User Interface, supporting interactive inputs and file inputs, was also developed to work on Ubuntu Systems.
  • Developed an algorithm for the before-stated functionalities, that gives accurate outputs for least information from the User.

Foot Mould Reconstructor Using Plantar Pressure

March 2018-May 2018

  • This project was aimed at developing a 3D Mould of the Foot, from the plantar pressure measurements from a platform.
  • The plantar pressure was measured, using a platform developed by another IIT Delhi Student, which was then used to build the 3D Mould of the foot sole.
  • The model was developed in the form of an STL file, that can directly be used for CAD Modelling and 3D Printing.
  • Perform various filtering operations on the data obtained from the platform, using the data to obtain an accurate measurement of the foot sole (using Gaussian filters for the purpose) and to render it in the form of an STL file.

Person Counter and Display Device
May 2017-September 2017

  • This project was developed as a part of the Design Innovation Summer Award (DISA) under Prof. Subodh Kumar.
  • Used OpenCV and Python to process images taken in a room to count the number of people in the specified range and to avoid occlusion.
  • Implemented on-site real-time processing using Beaglebone, interfaced with a camera.
  • The results were then sent via mail to a pre-specified authority.

Boids Flocking Simulation
March 2018-May 2018

  • The aim of the project is to develop a simulation for the starlings murmuration - Flocking Behaviour exhibited by birds.
  • The flocking behaviour can be expressed mathematically in terms of three basic rules establised by Craig W. Reynolds here.
  • The project implemented the three rules and more add ons and provided a Graphical Interface to observe the simulation.
  • The main challenge was to efficiently code the simulation for maximum number of boids, and to develop the appropriate graphical interface to display the simulation.

Mouse Positioning using VHDL
October 2017-November 2017

  • The project was done as part of COL215 course, under Prof. Anshul Kumar. Implemented a VHDL code to interface USB mouse with Basys 3 board using the USB HID port on the FPGA.
  • Processed the input stream from the mouse, as per a designed FSM to control the states and to exchange data with mouse at requisite states.
  • The coordinates received from the mouse was used to blink various LEDs as per the coordinates of the mouse.
